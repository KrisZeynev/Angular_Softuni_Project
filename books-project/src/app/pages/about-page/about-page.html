Welcome to my Angular Books Project!

Overview

The main purpose of this project is to showcase all the skills acquired during the Angular course at SoftUni.

The main functionalities of the platform include adding books and if the user is the creator of the book, they can edit it. Additionally, users can add comments.

Technologies Used

  • Angular – for speed and modularity
  • Custom CSS – for styling
  • SoftUni practice server – for backend
  • Angular Router – for client-side navigation
